**Descrição do Projeto – Sistema Bancário em Python**

Este projeto consiste no desenvolvimento de um sistema bancário simples utilizando a linguagem Python. O objetivo é simular operações básicas realizadas em um banco, permitindo ao usuário interagir com o sistema por meio de um menu no terminal.

O programa possibilita a realização das seguintes operações:

* **Depósito:** permite adicionar valores ao saldo da conta, desde que o valor informado seja válido.
* **Saque:** permite retirar valores do saldo, respeitando as regras de limite por saque, saldo disponível e limite máximo de saques diários.
* **Extrato:** exibe todas as movimentações realizadas durante a execução do programa, bem como o saldo atual da conta.
* **Encerramento do sistema:** opção para finalizar a execução do programa.

Durante o desenvolvimento foram aplicados conceitos fundamentais da programação em Python, como estruturas condicionais, laços de repetição, variáveis, manipulação de strings e tratamento básico de entradas do usuário.

O sistema foi projetado para garantir validações nas operações, evitando valores inválidos, saques acima do limite permitido ou acima do saldo disponível.

Este projeto tem como objetivo praticar a lógica de programação e a construção de aplicações simples em ambiente de terminal.
